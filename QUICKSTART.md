# MK AI OS — Quick Start

## เริ่มใช้งาน
1. เลือก Skill ที่ตรงกับงานจาก `skills/core/` หรือเรียก `Manager` เมื่อไม่แน่ใจ
2. หากเป็นงานของ MK Collection ให้โหลดกฎจาก `brands/mk-collection/SKILL.md` ก่อนเสมอ
3. เลือก Workflow จาก `workflows/`
4. ส่งต่อข้อมูลระหว่าง Skill ตาม `contracts/handoff-contract.md`
5. ตรวจ Quality Control ก่อนส่งมอบทุกครั้ง

## คำเรียกสั้น
- `Manager` — เลือก Skill และ Workflow
- `Planner` — วางแผน
- `Research` — วิเคราะห์ข้อมูล
- `Video` — ผลิต Storyboard และ Video Prompt
- `Review` — ผลิต Review Overlay และ Review Content
- `Copy` — เขียนข้อความ
- `Publish` — ตรวจความพร้อมโพสต์
- `Analytics` — วิเคราะห์ผลลัพธ์
- `MK Collection` — โหลดกฎเฉพาะแบรนด์ MK Collection

## หลักการทำงาน
- ข้อมูลครบแล้วเริ่มงานทันที
- ถามเฉพาะข้อมูลที่จำเป็นจริง
- ใช้ Skill ให้น้อยที่สุดที่ทำงานได้ครบ
- ไม่ทำขั้นตอนซ้ำ
- Brand Module เป็นแหล่งอ้างอิงกฎแบรนด์เพียงจุดเดียว
