# Review Production Workflow v2.0

## Trigger
Use for social-native product review visuals, benefit overlays, influencer-style layouts, and review poster/video keyframes.

## Stages

### 1. Review Creative Brief
Capture product, verified benefits, audience, platform, setting/reference, visual tone, number of shots, and text-density preference.

### 2. Storyboard Card + Lock
Design multiple review angles with varied compositions. Each shot contains one primary communication purpose and one concise benefit/message. Keep text light enough to remain readable. Apply final QC rules during creation. After approval, lock as the visual master.

### 3. Storyboard Split
Create one extraction instruction for clean separate 9:16 images. Preserve visual content, remove numbering/panel markers, and do not invent or redesign.

### 4. Review Overlay Design
Define only the overlay layer: typography hierarchy, bubbles, doodles, arrows, labels, stickers, spacing, safe zones, and visual rhythm. Do not repeat source attachment or split instructions.

### 5. Ready-to-Post Caption
Write a concise caption consistent with the approved benefits, tone, and platform. Avoid unsupported claims.

### 6. Final QC
Check benefit truthfulness, text hierarchy, readability, product visibility, layout balance, reference fidelity, brand consistency, and duplicated instructions.

## Completion Criteria
The user receives the storyboard/split/overlay/caption package with no redundant prompt-generation stage.
