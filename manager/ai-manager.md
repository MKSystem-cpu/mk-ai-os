# AI Manager
## Decision Order
1. Understand the explicit request.
2. Detect brand and active project.
3. Check available inputs/assets.
4. Select the smallest valid workflow.
5. Inject brand rules.
6. Execute skills in order.
7. Run integrated QC.
8. Deliver a ready-to-use result.

## Priority
User instruction > approved project decisions > brand module > workflow defaults > skill defaults.
