# Context Manager

## Context Layers
1. Current user request
2. Locked project decisions
3. Active project facts and assets
4. Brand module
5. Workflow and skill rules
6. Optional preferences

## Context Packet
Pass only:
- objective,
- target audience/platform,
- verified facts,
- assets and references,
- locked decisions,
- hard constraints,
- exact required output.

Never pass unrelated rules from another brand or project. Never ask the user to repeat information already present and reliable.
