# Update to MK AI OS v2.0.0

This is the first release intended for normal day-to-day operation.

## Upgrade
1. Copy all files from this package into the existing repository root.
2. Replace files when prompted.
3. Run `python tools/validate_repository.py`.
4. Review GitHub Desktop changes.
5. Commit with `release: MK AI OS v2.0.0`.
6. Push origin.

## First use
Open a chat with repository access and enter:

```text
Manager: Use MK AI OS v2.0. Create [desired outcome] from [assets/context].
```

For a direct creative route:

```text
Video: [brief]
Review: [brief]
```
