# Review Skill Workflow
1. Creative Brief: product truth, audience, platform, review purpose, references.
2. Storyboard Card + Lock: multiple review angles; integrate benefits and key messages into each shot; briefly state each shot's communication intent.
3. Storyboard Split Prompt: use approved Storyboard Card as sole master; extract clean 9:16 shots; remove numbering; no redesign.
4. Review Overlay Design: bubble typography, doodles, arrows, benefit labels, stickers, social review composition; do not repeat prior attachment instructions.
5. Optional review-video direction when requested.
6. Final QC: product accuracy, readable hierarchy, natural review tone, visual continuity, no exaggerated claims.
