# วิธีอัปโหลดขึ้น GitHub ผ่านเว็บไซต์

## สร้าง Repository ใหม่
1. เข้า GitHub และกดปุ่ม **New**
2. ตั้งชื่อ Repository เช่น `mk-ai-os`
3. เลือก Private หรือ Public
4. ไม่ต้องเลือกสร้าง README เพราะแพ็กนี้มีแล้ว
5. กด **Create repository**

## อัปโหลดไฟล์
GitHub Web ไม่รองรับการอัปโหลดโฟลเดอร์ ZIP แล้วแตกไฟล์ให้อัตโนมัติ

วิธีที่ง่ายที่สุด:
1. แตกไฟล์ `MK-AI-OS-v1.1.zip` ในคอมพิวเตอร์
2. เปิดโฟลเดอร์ `MK-AI-OS-v1.1`
3. ในหน้า Repository กด **Add file → Upload files**
4. ลากไฟล์และโฟลเดอร์ทั้งหมดภายในโฟลเดอร์นั้นลงในพื้นที่อัปโหลด
5. ใส่ Commit message: `Initial MK AI OS v1.1`
6. กด **Commit changes**

หากเว็บไม่ยอมลากโฟลเดอร์จำนวนมาก ให้ใช้ GitHub Desktop ซึ่งรองรับโครงสร้างทั้งหมดได้ง่ายกว่า
